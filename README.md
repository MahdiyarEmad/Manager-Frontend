# مارو منیجر - Marv Manager

سیستم مدیریت گارانتی و تعمیرات

## نصب و راه‌اندازی

```bash
# نصب پکیج‌ها
npm install

# اجرای سرور توسعه
npm run dev

# ساخت برای پروداکشن
npm run build
npm start
```

## ویژگی‌ها

- 👥 مدیریت پرسنل
- 📦 مدیریت محصولات
- 🖥️ مدیریت دستگاه‌ها
- 🔧 مدیریت تعمیرات
- 🧪 مدیریت تست‌ها
- 📜 استعلام گارانتی (عمومی)
- ⚙️ حساب کاربری

## تکنولوژی‌ها

- Next.js 15
- React 19
- TailwindCSS
- فونت IRANYekan

## ساختار پروژه

```
marv-manager/
├── app/
│   ├── login/          # صفحه ورود
│   ├── dashboard/      # داشبورد
│   ├── persons/        # مدیریت پرسنل
│   ├── products/       # مدیریت محصولات
│   ├── devices/        # مدیریت دستگاه‌ها
│   ├── repairs/        # مدیریت تعمیرات
│   ├── tests/          # مدیریت تست‌ها
│   ├── warranty/       # استعلام گارانتی
│   ├── account/        # حساب کاربری
│   ├── layout.jsx      # لایوت اصلی
│   ├── page.jsx        # صفحه اصلی
│   └── globals.css     # استایل‌ها
├── components/
│   ├── Sidebar.jsx     # منوی کناری
│   ├── DataTable.jsx   # جدول داده
│   ├── Modal.jsx       # مودال
│   └── Toast.jsx       # اعلان
├── lib/
│   ├── api.js          # کلاینت API
│   └── constants.js    # ثابت‌ها و برچسب‌ها
└── ...
```

## تنظیمات API

آدرس پیش‌فرض API: `http://localhost:8000/api`

برای تغییر آدرس API:
1. در صفحه ورود، روی "تنظیمات سرور" کلیک کنید
2. یا از صفحه حساب کاربری تغییر دهید
